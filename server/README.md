## Getting Started

### Prerequisites
* [Python](https://www.python.org/)
* [PIP](https://pypi.org/project/pip/)

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/bennaaym/zeta-basic.git
   ```
2. Install python packages 
   ```sh
   pip install -r requirements.txt
   ```
